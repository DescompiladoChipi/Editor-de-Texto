# Editor De Texto
Este editor de texto es la implementacion de lo aprendido en la clase del algoritmos durante el segundo semestre de 2025, dando paso a este proyecto final.

## Pantalla inicial
Al ejecutar nuestro codigo nos recibe una pantalla inicial con el logo de la Universidad Mariano Galvez de Guatemala, tambien encontraremos un texto nos indica que al presionar el boton ingresaremos al editor de texto, todo esto utilizando una paleta de colores acorde a la utilizada por la universidad.

## Barra de Menus
En la parte superior del programa encontraremos una barra de menu que incluye las opciones Archivo, Editar y Ayuda.

### Archivo
Al presionar la opcion archivo, obtendremos 6 opciones.
1. #### Nuevo
   Esta opcion nos permite generar un nuevo archivo de texto.
2. #### Abrir
   Esta opcion nos dara acceso a los archivos que existen en nuestro pc para abrir un archivo previamente guardado.
3. #### Guardar 
   La opcion Guardar nos permitira sobreescribir los cambios a un archivo existente.
4. #### Guardar como
   Al presionar esta opcion el programa nos dara la opcion de guardar el archivo como un archivo nuevo.
5. #### Buscar
   Con esta opcion podremos buscar una palabra o caracteres dentro del archivo de texto.
6. #### Salir
   Esta opcion cierra el programa por completo.

### Editar
Esta parte del menu nos presenta 7 opciones para editar el archivo de texto que tenemos abierto.
1. Deshacer
2. Rehacer
3. Copiar
4. Cortar
5. Pegar
6. Seleccionar todo

Esto nos permitira manipular nuestro archivo.

### Ayuda
La opcion ayuda nos muestra informacion sobre nuestro editor.
1. #### Informacion
   Esta opcion nos presenta informacion general acerca del editor, version, licencia, anio de creacion, etc.
2. #### Manual de usuario
   En este caso esta opcion te redigira a este manual.
3. #### Integrantes
   Esta opcion nos desplega una ventana con la informacion de los participantes
